# Description

This is an parser package for reading in data from XML files produced by FFI. It also allows us to insert data into a database and perform duplicate checks. Our organization needs to be able to transform these XML files into dataframes for a variety of purposes including the ETL process to insert data onto our server as well as QC the data.
